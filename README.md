Teamspeak3 Puppet Module
====================

Overview
--------

Installs and starts a teamspeak3 server

Requirements
------------

 - [puppet-archive](https://github.com/camptocamp/puppet-archive)

Usage
-----

Example:

    teamspeak3 { 'myts': }
